# 🧩 PTU Revamp (Customer App V12.30.0)

> Aim to simplify the PTU journey and reduce top-up steps for faster transactions and Enhancement on UI to avoid confusion between L2 KYC document uploading.

---

## 📌 1. Feature Summary

Brief Description:
_What is this feature and what does it do? (1–2 lines)_
> Simplifying PTU Journey and reduce top-up steps, Implement new core integration and Launch a PTU portal to manage denominations without app updates.

Goal / Value Proposition:
_What problem does it solve? What's the business/user value?_
> Expectation:
> - Improve customer satisfaction and user experience for all telco top-up transactions.
> - Enable future expansion to support telco bundles, plans and other denominations.
> - Establish a scalable and modern payment experience by revamping the PTU interface based on market benchmarks.

---

## 🛠 2. What's New

> Improve PTU Interface & Dynamic Control: Truemoney users can now enjoy an improved Phone Top-up experience, with enhanced backend dynamic controls to manage and optimize best-selling PTU values.
> - Each of the PINLESS top-up now is possible to configure from dynamic back office portal.

> Improve L2 KYC document interface: Improved document display for clearer visibility and easier indentation of document types during the registration process.

> Minor Bugs Improvement:
> - Scanning KHQR from other bank improvement
> - Resolve online payment got return back home screen from QR
> - TrueMoney Nearby responsive
> - Money Drop Enhancement -  Prevent multiple money boxes claimed per session.
